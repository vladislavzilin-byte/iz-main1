# IZ Booking v2 (Vite + React, localStorage)

Простая запись к мастеру: вход/регистрация (с Instagram), календарь c выбором даты на месяц вперёд,
рабочие дни по умолчанию — все 7, админ-панель со списком записей и настройками.

## Старт
```bash
npm i
npm run dev
```
Сборка:
```bash
npm run build
npm run preview
```

## Что нового в v2
- Рабочие **все 7 дней** по умолчанию.
- **Календарь-пикер на месяц** вперёд (перелистывание месяцев, сетка дат).
- Поле **Instagram** в регистрации, отображается в админке.
- Небольшой визуальный полиш (hover/focus).

## Интеграция
Перенесите компоненты `Auth`, `Calendar`, `Admin`, библиотеку `src/lib/storage.js` и стили `src/styles.css`.
